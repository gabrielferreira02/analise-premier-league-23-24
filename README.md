Projeto de análise de dados da Premier League utilizando Python e Jupyter Notebook, com foco em identificar os principais destaques individuais da temporada por meio de métricas ofensivas, disciplinares e de criação de jogadas.

# Principais Análises Desenvolvidas

O projeto extrai e organiza estatísticas para responder perguntas como:

- Quem são os artilheiros da liga

- Jogadores com mais assistências

- Atletas com mais cartões amarelos e vermelhos

- Quem mais participou de gols (gols + assistências)

- Jogadores que mais carregaram a bola ao ataque

- Quem realizou mais passes progressivos / para frente

- Quem possui maior expectativa de participação em gols
